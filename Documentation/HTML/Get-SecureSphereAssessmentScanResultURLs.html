Get-SecureSphereAssessmentScanResultURLs

SYNOPSIS

Returns assessment scan results locations for a single scan.

SYNTAX

Parameter Set 1 (Default)

Get-SecureSphereAssessmentScanResultURLs -ScanName <String> [<CommonParameters>]

DESCRIPTION

Returns assessment scan results locations for a single scan.

EXAMPLES
