# Get-SecureSphereAllCorrelationPolicies

## SYNOPSIS
Retrieves the names of all web correlation policies.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllCorrelationPolicies [<CommonParameters>]
```

## DESCRIPTION
Retrieves the names of all web correlation policies.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllCorrelationPolicies
```
