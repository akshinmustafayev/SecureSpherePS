# Get-SecureSphereAllFirewallPolicies

## SYNOPSIS
Return all Firewall Policies.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllFirewallPolicies [<CommonParameters>]
```

## DESCRIPTION
Return all Firewall Policies.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereFirewallPolicy -PolicyName "policy 1"
```
