# Get-SecureSphereAllWebProfilePolicies

## SYNOPSIS
Retrieves the names of all web profile policies.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllWebProfilePolicies [<CommonParameters>]
```

## DESCRIPTION
Retrieves the names of all web profile policies.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllWebProfilePolicies
```
