# Get-SecureSphereAllWebActionSets

## SYNOPSIS
Retrieves the names of all action sets.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllWebActionSets [<CommonParameters>]
```

## DESCRIPTION
Retrieves the names of all action sets.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllWebActionSets
```
