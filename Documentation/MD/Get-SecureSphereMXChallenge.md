# Get-SecureSphereMXChallenge

## SYNOPSIS
Returns the challenge of the MX that the installer resides on.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereMXChallenge [<CommonParameters>]
```

## DESCRIPTION
Returns the challenge of the MX that the installer resides on.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereMXChallenge
```
