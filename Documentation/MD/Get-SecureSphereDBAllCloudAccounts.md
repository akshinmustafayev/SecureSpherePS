# Get-SecureSphereDBAllCloudAccounts

## SYNOPSIS
Get all the display names of the cloud accounts.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereDBAllCloudAccounts [<CommonParameters>]
```

## DESCRIPTION
Get all the display names of the cloud accounts.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereDBAllCloudAccounts
```
