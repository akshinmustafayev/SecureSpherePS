# Get-SecureSphereAllAssessmentScanNames

## SYNOPSIS
Returns the names of all the assessment scans.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllAssessmentScanNames [<CommonParameters>]
```

## DESCRIPTION
Returns the names of all the assessment scans.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllAssessmentScanNames
```
