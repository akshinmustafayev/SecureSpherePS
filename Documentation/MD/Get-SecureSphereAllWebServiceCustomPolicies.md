# Get-SecureSphereAllWebServiceCustomPolicies

## SYNOPSIS
Returns a list of names of all Web Service Custom Policies in the system.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllWebServiceCustomPolicies [<CommonParameters>]
```

## DESCRIPTION
Returns a list of names of all Web Service Custom Policies in the system.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllWebServiceCustomPolicies
```
