# Get-SecureSphereAllClassificationScanNames

## SYNOPSIS
Returns the names of all the classification scans.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllClassificationScanNames [<CommonParameters>]
```

## DESCRIPTION
Returns the names of all the classification scans.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllClassificationScanNames
```
