# Get-SecureSphereADCTimeStamp

## SYNOPSIS
Retrieves the timestamp of the latest ADC update.

## SYNTAX

### Parameter Set 1
```
Get-SecureSphereADCTimeStamp [<CommonParameters>]
```

## DESCRIPTION
Retrieves the timestamp of the latest ADC update.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereADCTimeStamp
```
