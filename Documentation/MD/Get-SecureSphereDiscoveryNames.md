# Get-SecureSphereDiscoveryNames

## SYNOPSIS
Returns all discovery scan names.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereDiscoveryNames [<CommonParameters>]
```

## DESCRIPTION
Returns all discovery scan names.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereDiscoveryNames
```
