# Get-SecureSphereAllMXs

## SYNOPSIS
Returns a list of all the mxs in the system.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllMXs [<CommonParameters>]
```

## DESCRIPTION
Returns a list of all the mxs in the system.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllMXs
```
