# Get-SecureSphereAllClassificationProfileNames

## SYNOPSIS
Returns the names of all the classification profiles.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllClassificationProfileNames [<CommonParameters>]
```

## DESCRIPTION
Returns the names of all the classification profiles.
