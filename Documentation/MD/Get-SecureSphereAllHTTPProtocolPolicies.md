# Get-SecureSphereAllHTTPProtocolPolicies

## SYNOPSIS
Retrieves the names of all http protocol policies.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllHTTPProtocolPolicies [<CommonParameters>]
```

## DESCRIPTION
Retrieves the names of all http protocol policies.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllHTTPProtocolPolicies
```
