# Get-SecureSphereAllIPGroups

## SYNOPSIS
Get a list of existing IP Group names.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllIPGroups [<CommonParameters>]
```

## DESCRIPTION
Get a list of existing IP Group names.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllIPGroups
```
