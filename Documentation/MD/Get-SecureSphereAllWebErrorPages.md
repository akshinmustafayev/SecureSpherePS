# Get-SecureSphereAllWebErrorPages

## SYNOPSIS
Get a list of existing Web Error Page names.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereAllWebErrorPages [<CommonParameters>]
```

## DESCRIPTION
Get a list of existing Web Error Page names.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllWebErrorPages
```
