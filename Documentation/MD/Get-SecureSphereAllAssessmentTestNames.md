# Get-SecureSphereAllAssessmentTestNames

## SYNOPSIS
Returns the names of of all the DAS assessment tests.

### Parameter Set 1 (Default)
```
Get-SecureSphereAllAssessmentTestNames [<CommonParameters>]
```

## DESCRIPTION
