# Get-SecureSphereAllRunningJobs

## SYNOPSIS
Returns a list of all the jobs that are currently executed.

## SYNTAX

### Parameter Set 1
```
Get-SecureSphereAllRunningJobs [<CommonParameters>]
```

## DESCRIPTION
Returns a list of all the jobs that are currently executed.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-SecureSphereAllRunningJobs
```
