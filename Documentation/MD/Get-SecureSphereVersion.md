# Get-SecureSphereVersion

## SYNOPSIS
Returns the version of the server.

## SYNTAX

### Parameter Set 1 (Default)
```
Get-SecureSphereVersion [<CommonParameters>]
```

## DESCRIPTION
Returns the version of the server.

### EXAMPLE 1

```powershell
Get-SecureSphereVersion
```
